
# Changelog

## Modifications

- Improved readme, added symlink
- Releases now include a variant per platform
- Toned down the default probability to add various consumables
- Included compatibility with IWDEE, adding stock to various merchants and creatures
- Fixed bug with Elixir of Mystical Cultivation showing a console line about the effects fading at the moment of consumption inappropriately
- Fixed some oils being granted to non-humanoids when it was meant to be given (automatically, based on chance, only to humanoids)
- Goblins are always excluded from being given consumables, as they are used all over the place in IWDEE and BG2EE.
